Судоку — логическая головоломка. Цель — заполнить все поле числами. Каждое число может встречаться только один раз в каждом столбце, строке и подразделе.

Судоку (PFA) имеет три режима игры:
1. игровое поле 6x6 с подразделами 2x3
2. игровое поле 9x9 с подразделами 3x3
3. игровое поле 12x12 с подразделами 3x4

Для каждого игрового режима существует четыре различных уровня сложности, которые измеряются не количеством предустановленных значений, а скорее стратегиями решения, необходимыми для решения игры. Генератор всегда пытается предоставить игру с минимальным количеством предустановленных значений, необходимым для решения игры. Кроме того, пользователи могут создавать свои собственные судоку произвольной сложности для каждого из трёх игровых режимов.

Судоку можно делиться с другими пользователями, используя доступную функциональность импорта и экспорта. Ещё одной возможностью от сообщества является «Судоку дня», которое генерируется каждый день и одинаково для каждого пользователя приложения.

Приложение можно использовать как в светлом, так и в тёмном режиме.

Наше приложение Privacy Friendly Sudoku отличается от других приложений двумя аспектами:
1. Приложение Судоку не использует никаких разрешений.
2. Приложение Судоку полностью без рекламы.
Многие другие бесплатные приложения в магазине Google Play слепят раздражающей рекламой, а также сокращают срок службы батареи.
